void Compass::Calibrate ( int  numberSeconds)

Calibrates the MAG3110. Spin the device several times full circle for the duration of time specified in the parameter. This reads data coming in, gathers minimum and maximum data values for each axis, then takes the difference of max-min for each axis. This is used to scale the data read from the magnometer to generate degrees pointed from north.

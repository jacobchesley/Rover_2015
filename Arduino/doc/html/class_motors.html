Detailed Description

The L298N Motor Controller Class for Arduino.

Author
Jacob Chesley
Date
3/12/2015

This controls a set of 2 motors using the L298N motor controller

Included several functions that are useful when motors are facing opposite directions and paralled and coplanar with eachother

For use with Arduino
